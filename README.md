# Thiva Engineering & Construction - Frontend

A high-performance, SEO-optimized business platform built with **Next.js**, featuring a decoupled architecture powered by a **Laravel API**. This project demonstrates modern frontend best practices, including Server-Side Rendering (SSR), Type-safety, and automated testing.

---

## Key Features & Implementation

### Advanced SEO & Performance
* **Custom Meta Management:** Manually configured unique titles, meta descriptions, and header tags for every page to ensure maximum relevance for search engine crawlers.
* **Search Engine Indexing:** Implemented a custom `sitemap.xml` and `robots.txt` to optimize site architecture discovery and crawl budget.
* **Core Web Vitals:** Optimized for performance with `next/image` for lazy loading and Tailwind CSS for minimal CSS bundle sizes.
* **Server-Side Rendering (SSR):** Leveraged Next.js to ensure all content is pre-rendered and visible to search engines on the first request.

### Architecture & State
* **Type-Safe Development:** Full TypeScript implementation to catch errors at compile-time and improve developer experience.
* **Component-Driven UI:** Built using a modular approach with reusable UI components, ensuring consistency and easy maintenance.
* **Headless Integration:** Consumes data from a custom Laravel REST API, demonstrating a clear separation of concerns between frontend and backend.

### Quality Assurance
* **Unit Testing:** Comprehensive test suite using **Jest** and **React Testing Library** covering critical components and page layouts to ensure stability.

---

## Tech Stack

* **Framework:** [Next.js](https://nextjs.org/) (React)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)

* **Testing:** [Jest](https://jestjs.io/) & React Testing Library
* **Backend:** [Laravel 12 (REST API)](https://laravel.com/docs/12.x/)
